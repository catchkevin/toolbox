
# git status info
It will tell you: "Your branch is ahead of 'origin/main' by x commits."
That is your signal that you have x "Save Points" sitting on your computer that aren't in the cloud yet.

```powershell
git status
```

# .
In your terminal, there is a legendary command that creates a "mini-map" of your project.
Developers often remember it by the acronym ADOG: git log --all --decorate --oneline --graph

```powershell
git log --all --decorate --oneline --graph
```
